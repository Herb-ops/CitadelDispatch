# Satoshi's Observatory

Satoshi's Observatory is a Bitcoin technical-social intelligence and education platform.

Its near-daily publication is **The Citadel Dispatch**: a source-forward briefing that translates high-signal Bitcoin development activity into clear, neutral public explanation.

## Mission

Bridge the gap between:

- **Public Bitcoin commentary** (often repetitive and slogan-driven), and
- **Developer/protocol discourse** (high-signal but fragmented and socially hard to access).

The Citadel Dispatch exists to represent serious positions fairly, surface what is actually happening, and route readers to primary sources so they can verify and learn for themselves.

## Editorial Principles

1. **Neutral, not detached**: report disagreements clearly without rhetorical side-taking.
2. **Source-forward**: prioritize direct links to PRs, BIPs, mailing list posts, specs, release notes, and reproducible data.
3. **Signal over noise**: include only developments with technical, governance, coordination, or infrastructure significance.
4. **Multi-perspective synthesis**: capture legitimate disagreements without flattening nuance.
5. **No cliché layer**: avoid price theater, tribal propaganda, culture-war framing, and empty talking points.

## Product Structure

- **Institution / archive / research layer**: Satoshi's Observatory
- **Transmission layer (near-daily brief)**: The Citadel Dispatch

The Dispatch uses a modular issue format:

- **Fixed fields**: title + one-sentence abstract
- **Flexible modules**: selected by the day’s signal rather than a rigid section order

See `docs/` for operating specs.
